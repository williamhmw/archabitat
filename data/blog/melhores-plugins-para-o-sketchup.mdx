# Melhores plugins para SketchUp

![Plugins Sketchup](/static/images/melhores-plugins-para-sketchup/plguins-capa.jpg)

Fizemos um compilado com os plugins mais utilizados pelos usuários do SketchUp.

# RoundCorner

![RoundCorner](/static/images/melhores-plugins-para-sketchup/RoundCorner___Interactive_selection_4.png)

Esse plugin realiza o arredondamento das arestas e cantos dos modelos 3D no Sketchup ao longo de um perfil 2D, com 3 modos: cantos arredondados, cantos agudos e chanfro.

Para fazer o download [clique aqui](https://sketchucation.com/pluginstore?pln=RoundCorner).

# Joint Push Pull Interactive

![Joint Push Pull Interactive](/static/images/melhores-plugins-para-sketchup/JointPushPull_Summary_of_Tools.png)

O Joint Push Pull Interactive executa extrusões nas faces dos modelos 3D.

Para fazer o download [clique aqui](https://sketchucation.com/pluginstore?pln=JointPushPull).

# 1001bit

![1001bit](/static/images/melhores-plugins-para-sketchup/1001bit_roof01.jpg)

Através do 1001bit você pode criar facilmente modelos 3D de escadas, escadas rolantes, paredes, portas e janelas, vigas de telhado automáticas digitando apenas as dimensões e alguns cliques do mouse.

Para fazer o download [clique aqui](http://www.1001bit.com/freeware/index.html).

# Floor Generator

![Floor Generator](/static/images/melhores-plugins-para-sketchup/floorgenerator.jpg)

O Floor Generator criar em poucos passos pisos, decks, tijolos, e muitos outros recursos que exigem paginação. é um dos plugins mais utilizados para trabalhar com paginação de pisos e azulejos.

Para fazer o download [clique aqui](https://download602.mediafire.com/3pdz1ah0skbg/okl51f4aa7gfl4d/SDM_FloorGenerator.rbz).

# Solar North

![Solar North](/static/images/melhores-plugins-para-sketchup/screenshot.png)

O Solar North exibe e altera a direção norte solar, usada para sombras. É útil para estudos de sombra, especialmente quando os eixos do modelo por qualquer motivo não se alinham com as direções cardeais.

Para fazer o download [clique aqui](https://download602.mediafire.com/3pdz1ah0skbg/okl51f4aa7gfl4d/SDM_FloorGenerator.rbz).
