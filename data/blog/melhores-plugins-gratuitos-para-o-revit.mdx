# Melhores plugins gratuitos para o Revit

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/Grevit.png)

Na época da faculdade, a descoberta de um programa capaz de gerar cortes e fachadas automáticas foi uma ótima oportunidade para começar a conhecer o mundo BIM com o Revit, nesse tempo de pesquisa alguns plugins ajudaram a ganhar tempo para entrega dos projetos.

Pensando nisso preparei uma lista com os melhores plugin para o Revit, que podem te ajudar em algum momento da concepção do seu projeto.

---

# 1 – DigiPara Elevatorarchitect

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/digipara.png)

Esse plugin foi desenvolvido especialmente para modelos BIM 3D de elevadores pela empresa DigiPara AG, através dele temos uma solução completa para inserir diversos modelos de elevadores, em uma tela inserimos alguns dados para auxiliar na escolha ideal dos modelos, onde queremos inserir e o plugin faz todo o processo de forma automática, criando os cortes nas lajes ou o poço do elevador. O site possui uma documentação completa de como utilizar o plugin caso necessário, além de vídeos com tutoriais. Para acessar a página de downloads ou conhecer o plugin clique [aqui](https://www.digipara.com/products/digipara-elevatorarchitect/).

# 2 – Grevit

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/grevit-2.png)

O Grevit é um plugin capaz de transportar modelos produzidos no Grasshopper ou SketchUp diretamente para o Revit em um modelo BIM, suportando os principais elementos como paredes, vigas, colunas, componentes simples, componentes adaptáveis, topografias entre vários outros. Não é sobre somente exportar e importar os modelos mas transportar de forma nativa esses modelos para o Revit, sem necessidade de redesenhar partes do modelo. Para realizar o download de forma gratuita clique [aqui](https://grevit-dev.github.io/Grevit/).

# 3 – Augin

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/augin.png)

Plugin gratuito para gerar um modelo de realidade aumentada, comentada por nós nesse post, através dele podemos exportar para a plataforma do aplicativo nosso projeto em 3D para ser visualizado em realidade aumentada, para acessar a página especial do plugin clique [aqui](http://pauluzzi.rds.land/plugin-revit).

# 4 – ONBOX

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/augin.png)

Plugin gratuito disponível na loja de aplicativos da Autodesk, desenvolvido em 2014 como comandos de repetições foi lançado novamente como um plugin do Revit da Autodesk, como foco em modelagem, auxiliando na criação de componentes estruturais, topografias através de nuvens de pontos e o gerenciamento de conteúdo o plugin se tornou um dos mais utilizado no Brasil e em países pelo mundo. Para acessar a página do plugin e realizar o downlaod clique [aqui](https://apps.autodesk.com/RVT/en/Detail/Index?id=7481675320348789487&appLang=en&os=Win64).

# 5 – Export-Import Excel

![Plugins Revit](/static/images/melhores-plugins-gratuitos-para-o-revit/Export-Import-Excel.png)

Plugin gratuito disponível na loja de aplicativos da Autodesk, com foco em planejamento e documentação. Importe e exporte dados do modelo BIM no Revit e atualize os dados em tempo real entre a planilha e o modelo. Para acessar a página do plugin e realizar o downlaod clique [aqui](https://apps.autodesk.com/RVT/pt/Detail/Index?id=6290726048826015851&appLang=en&os=Win64).
