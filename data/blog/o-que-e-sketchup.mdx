# O que é Sketchup

![Sketchup](/static/images/o-que-e-sketchup/cinema-4d-capa)

O SketchUp é um software de modelagem 3d de propriedade da Trimble Navigation. O software foi adquirido pela empresa Google em 2006 e vendida posteriormente em 2012 para a Trimble.

# Software intuitivo

O programa tem uma maneira diferente de modelar comparado a outros softwares de modelagem 3D, com uma eficiência nas modelagens ele se tornou uma das principais ferramentas utilizadas por profissionais da área.

As ferramentas são bem simples e intuitivas como desenhar uma linha ou um quadrado, fazer um arco e depois puxá-los para cima para desenhar uma parede, com três comandos o usuário já tem a modelagem de uma parede no seu projeto. Essa eficiência e facilidade para modelar chamam a atenção de iniciantes no mundo 3D e se torna uma boa oportunidade para entrar nesse mercado.

As ferramentas complementares do software ainda permitem realizar a documentação do projeto de forma dinâmica no estilo desejado.

# 3D Warehouse

Há ainda um recurso extremamente prático para os profissionais que não podem ficar horas modelados móveis e o entorno do seu projeto. A 3D Warehouse pode ser acessada através de um site e é incorporado ao software, o usuário pode importar modelos prontos direto para dentro do programa.

São diversas fabricantes de todos os tipos de materiais como pisos, utensílios de cozinha, eletrodomésticos, entre outros que disponibilizam seus produtos já modelados na biblioteca da 3D Warehouse.

São objetos que vão de um parafuso e prego até uma casa pronta, com toda a estrutura e mobílias prontas.

# Recursos

Alguns projetos exigem recursos mais avançados como o desenho de topografias, geolocalização do projeto e a escolha de materiais para os produtos, no SketchUp á um leque grande de possiblidades para esses casos, todo muito intuitivos.

Outra ferramenta em destaque é a possibilidade dinâmica de alterar as medidas e volumes da modelagem

Caso você ainda necessite de ferramentas mais poderosas como um programa para renderizar ou desenhar uma escada complexa, há plugins que podem ser instalados no programa para tornar o uso mais eficiente, como o V-Ray para produzir imagens realistas ou o 1001 bit Tools.

Há também plugins para verificar o conforto térmico dentro de um ambiente ou onde haverá sombras de acordo com o volume do modelo projetado.

# Vantagens do SketchUp

Por ser um software extremamente conhecido e aceito pelos profissionais há uma infinidade de tutoriais e artigos que ensinam a usar o programa além de uma comunidade engajada.

O arquivo gerado pelo Sketchup com a modelagem é compatível com diversos outros softwares da área. Isso amplia ainda mais as possiblidades de criação dos seus usuários.

# SketchUp online gratuito

Há uma versão do programa que pode ser utilizada direto pelo navegador, sendo possível modelar e visualizar os modelos sem baixar nada, de forma gratuita.

É uma versão básica do SketchUp, com uma modelagem mais simples e leve, ideal para modelar ideias rápidas que podem vir a surgir.

# Como adquirir o SketchUp

Caso você deseje adquirir o programa há alguns planos de assinaturas disponíveis que podem várias de acordo com o seu tipo de trabalho e recursos desejados. O SketchUp pode ser testado por até 30 dias de graça antes de adquirir o plano desejado.
