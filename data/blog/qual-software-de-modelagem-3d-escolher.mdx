# Qual software de modelagem 3D escolher

![Modelagem 3D](/static/images/qual-software-de-modelagem-3d-escolher/3dsmax.jpg)

Para construirmos nosso projeto em 3D precisamos de um software para fazer a modelagem. Existe uma grande variedade de softwares no mercado para modelagem 3D, nessa lista separei os principais que você deveria considerar, cada uma delas possuem ferramentas em comum e ferramentas específicas para suas propostas.

# 1 – 3ds Max

![Modelagem 3D](/static/images/qual-software-de-modelagem-3d-escolher/3dsmax-capa.jpg)

Provavelmente o mais conhecido da lista, o software 3ds Max produzido pela Autodesk é utilizado massivamente pela indústria, freelancers e amantes do 3D, ele pode ser utilizado para modelagem, animação, renderização e visualização 3D. Um programa completo independentemente da proposta que você tem para utiliza-lo, desde uma animação dos seus projetos ao render.

Com diversos plugin disponíveis no mercado, seu grande diferencial é a quantidade de tutoriais, curso, recursos e suporte da comunidade, principalmente na fase de aprendizado.

Apesar de ser um software pago, com a qualidade e recursos que o programa entrega pode vir a ser um investimento interessante para quem trabalha com o 3D, principalmente para que é freelancer que tem uma versão Indie do software com valores bem inferiores.

Assim como outros softwares da Autodesk, há também uma versão para estudante totalmente gratuita, basta comprovar o vínculo com uma instituição de ensino e aderir ao programa.

# 2 - Blender

![Modelagem 3D](/static/images/qual-software-de-modelagem-3d-escolher/blender.jpg)

O software de código aberto totalmente gratuito Blender esta tomando conta do cenário 3D nos últimos anos, com uma fase de desenvolvimento cheio de novos recursos e aprimoramentos de ferramentas, o Blender vem ganhando mais espaço no meio 3D.

Com recursos de modelagem, rigging, animação, renderização, entre outros é um software excelente para quem está começando no mundo 3D. Assim como o 3ds Max, no Blender também é possível adicionar plugins ao software de acordo com a necessidade, seja um renderizador ou um plugin de modelagem para arquitetura.

Talvez uma das maiores vantagens dentre todos os programas seja o estágio de desenvolvimento do Blender, quase a todo momento os usuários podem acompanhar através do site oficial as últimas notícias em relação a seu desenvolvimento e projetos futuros.

Caso tenha dúvidas sobre ferramentas e como utilizá-lo para arquitetura basta uma busca rápida na internet. São vários os tutoriais e recursos disponíveis para iniciantes, desde grupos no Facebook a cursos na Udemy, além da documentação no site oficial.

# 3 – SketchUp

![Modelagem 3D](/static/images/qual-software-de-modelagem-3d-escolher/sketchup.jpg)

Talvez um dos softwares de modelagem mais utilizados por arquitetos seja o SketchUp. Com ferramentas extremamente fáceis de usar e disponibilização de muito conteúdo na internet, o software tem uma curva de aprendizado muito rápida.

Umas das vantagens é a facilidade de achar produtos reais na biblioteca oficial do programa. Lá é possível achar de tudo, desde uma geladeira de determinada marca a um carro para compor sua imagem, tudo já modelado, basta fazer o downlaod e importa-lo no seu projeto.

Há ainda ferramentas de documentação para montagem de pranchas. O software também possui extensões que aumentam ainda mais a velocidade com que os projetos são desenvolvidos.

# Cinema 4D

![Modelagem 3D](/static/images/qual-software-de-modelagem-3d-escolher/cinema-4d.jpg)

Caso o objetivo seja modelar um ambiente 3D e anima-lo posteriormente a melhor ferramenta para isso talvez seja o software Cinema 4D. Com uma curva de aprendizado fácil de aprender para iniciantes e extremamente poderosa chama a atenção.

Com tutoriais e uma comunidade engajada e qualificada é possível extrair ótimos resultados com as ferramentas disponíveis, com suas Interfaces práticas, sistema de ajuda e um software intuitivo e estável auxiliam no desenvolvimento de projetos a longo prazo.
