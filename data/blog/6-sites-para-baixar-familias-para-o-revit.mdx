# Introdução

![BIM](/static/images/fabricantes-brasileiras-com-biblioteca-em-bim/conexoes-tigre.PNG)

Durante o desenvolvimento de um projeto em BIM, chega o momento de colocar algumas famílias (ou blocos) no projeto. Muitas das vezes o objeto que necessitamos já existe em algum lugar na internet, sem necessidade de criar do zero. Para ajudá-los nessa pesquisa separamos 6 sites para baixar famílias gratuitamente.

# 1 – BIM Object

O BIM Object é um dos maiores sites de famílias em BIM do mundo, a grande maioria dos blocos são criados pelas próprias fabricantes dos produtos, isso muitas vezes aumenta o nível de detalhe e informações que as famílias carregam.

Aproveitando a repaginada recente do site, somos capazes de fazer uma pesquisa pelo nome dos produtos ou pelas categorias. Podemos ainda filtrar os resultados com base na localização do país em que os produtos estão disponíveis ou o tipo de arquivo que queremos, indo desde um arquivo DWG para AutoCAD, ou um JPG até um arquivo para Revit ou ArchiCAD.

Para facilitar ainda mais, o site tem suporte para o idioma Português do Brasil, basta fazer o cadastro, selecionar os produtos e fazer o Download.

# 2 – Família BIM

O site Família BIM é nativamente brasileiro, foi concebido por um time com o intuito de montar uma base de dados com famílias elaboradas pela equipe. As famílias podem ser enviadas pelos visitantes do site, ser sugeridas pelos proprietários de famílias ou pelas próprias fabricantes, criando assim uma solução para buscas de famílias.

Para realizar o download basta fazer um cadastro simples, selecionar o arquivo e baixar.

# 3 – Vitrine Revit

O site Vitrine Revit é uma empresa criada pela arquitetura Larissa Guerra, nele são disponibilizados diversas famílias voltadas para o detalhamento de interiores, trabalhando com mobiliários paramétricos.

É possível acessar as famílias por meio de uma biblioteca onde selecionamos a categoria e visualizamos os modelos. Basta fazer um cadastra e baixar gratuitamente.

# 4 – NBS National BIM Library

A National BIM Library é uma biblioteca online de modelagem de informações de construção (BIM), com uma coleção extensa de ótima qualidade com objetos genéricos e de fabricantes, são desde sistemas de tecidos a objetos mecânicos e elétricos. Todos atendendo a uma especificação internacional, utilizada no Reino Unido chamada NVS BIM Object Standard.

É possível fazer a pesquisa dos objetos diretamente do site da NBS, por nome do objeto ou do fabricante, ainda é possível aplicar um filtro com as categorias dos objetos.

O site também disponibiliza um plugin para download gratuitamente, com funções que agilizam a inserção dos objetos diretamente dentro do Revit. Existem também ferramentas com atualizações dos modelos inseridos e especificações dos objetos.

O padrão de objeto NBS BIM
Esse padrão de qualidade define requisitos mínimos para os objetos que são as informações nele contidas, geometria, comportamento e a apresentação em BIM, isso oferece qualidade e consistência para utilização em grandes projetos em colaboração com diversas indústrias da construção.

# 5 – BIM Store

A BIM Store também é uma empresa do Reino Unido, focada em alimentar uma biblioteca com diversas famílias para o Revit por meio de fornecedores, usuários e pela equipe da empresa.

Muitas das famílias disponíveis no site são de excelente qualidade, com diversas informações seguindo alguns requisitos mínimos de informação. As famílias foram construídas por profissionais da área de arquitetura e com amplo conhecimento na área BIM. Caso tenha dúvidas sobre alguns dos modelos, ou deseje solicitar alguma família de determinado fornecedor, é possível abrir um tópico no fórum e pedir ajuda a empresa.

Para realizar o download das famílias basta fazer um cadastro gratuitamente. Por meio do campo da pesquisa podemos informar o nome do produto, aplicar alguns filtros se necessário e buscar o objeto desejado.

As características que facilitam a navegação do site são, a facilidade de encontrar os produtos e excelentes recursos visuais que ajudam a identificar as famílias.

Caso deseje também é possível fazer o download de m plugin próprio da BIM Store e utilizá-lo diretamente dentro do Revit, facilitando ainda mais a busca pelas famílias.

# 6 – Revit City

O Revitcity surgiu no ano de 2003, possui milhares de modelos de blocos disponíveis gratuitamente para download por meio de um cadastro no site.

Dentre todas as opções acima, é o que tem um design um pouco mais antigo, o que acaba dificultando mais a localizar os objetos.

Por não ter um controle exato sobre os modelos disponíveis, não é recomendado utilizar a maioria das famílias para extrair algum tipo de informação.

Apesar desses pontos ele possui um Fórum muito ativo, ideal para tirar dúvidas quanto as modelos e sobre o próprio software.
