# O que é Blender e como usá-lo

![Blender](/static/images/o-que-e-blender-e-como-utilizalo/tiger-blender.jpg)

O programa de modelagem 3D Blender, antigamente conhecido como blender3d é um software de código aberto desenvolvido pela Blender Foundation. O software abrange diversas ferramentas sendo elas: modelagem, animação, texturização, composição, renderização e edição de vídeo. Todas elas são capazes de trabalhar em conjunto, a ideia é atender os requisitos de todas as etapas da linha de produção de um conteúdo que começa com a modelagem até a entrega do vídeo.

> Por ser um aplicativo multiplataforma, o Blender roda em sistema Linux, macOS e nos sistemas Windows.

# Ferramentas

Há ferramentas avançadas que só são encontradas em programas pagos como o sistema de fluídos e partículas. Algumas principais funcionalidades são:

- Integração com ferramentas essenciais como Modelagem, animação e simulações;
- Arquitetura 3D de alta qualidade, com fluxos de trabalho simples e preciso;
- Suporte ativo da comunidade;
- Possui um arquivo executável muito leve comparado ao número de ferramentas que possui, sendo possível executá-lo de forma portátil com um pendrive por exemplo.

A linguagem do programa é o Python, o que significa que podemos rodar scrpts para realizar tarefas complexas, assim como desenvolver ferramentas para usos específicos.

O Blender suporta diversos tipos de arquivos de outros softwares tridimensionais como 3ds Max, Autodesk fbx, COLLADA, SketchUP e muitos outros através de plugin.

A partir de 2018 o software passou por uma repaginação de todo visual e lançamento de diversos novos recursos com um plano de desenvolvimento acelerado, a partir da versão 2.80. A partir dessa versão a atualizações de recursos quase que mensais, tornando o software extremamente competitivo na sua área.

O software possui 2 renderizadores nativos para fazer a finalização das modelagens ou animações, o EVEE com visualização em tempo real e o Cycles com simulação real da luz.

# Plugins

O Blender suporta a adição de plugins graças a linguagem utilizada pelo programa, o python, podemos instalar plugins que realizam tarefas mais complexas de forma autônoma ou que adicionam novos recursos.

Apesar de ser um software gratuito ele pode ser utilizado para grandes produções, para isso a [Blender Foundation](https://www.blender.org/foundation/) mantem em desenvolvimento curtas metragens demostrando a capacidade de produção do programa.
