# O que é modelagem 3D

![Modelagem 3D](/static/images/o-que-e-modelagem-3d/Blender-capa-1.png)

A modelagem 3D é uma técnica de computação gráfica definida para representar determinado objeto de forma tridimensional, essa representação tem o nome de modelagem 3D, caso seja sua intenção há a possibilidade de imprimir esse modelo através de uma impressora 3D e trazê-lo para o mundo real.

Pra criar um modelo é necessário um software específico. Esses modelos podem ser utilizados profissionalmente para fins comerciais como arquitetura, filmes, jogos, publicidade, entre uma infinidade de possibilidades. Para visualizarmos o resultado desses modelos fazemos o processo de renderização 3D.

# Modelagem de curvas

Para definir as superfícies utilizamos curvas, cada ponto que colocamos é um ponto de controle na qual ajustamos a sua trajetória até ficar na forma que desejamos. Alguns exemplos de curvas presentes nos softwares são as splines, geometrias primitivas e NURBS.

# Modelagem poligonal

Para criar a malha do modelo é utilizado linhas que se conectam no espaço 3D, a conexão dessas linhas criam formas. É o principal método utilizado para criar os modelos devido a facilidade para o artista e o poder de processamento menor que o computador precisa processar.

Um ponto a desejar dessa modelagem são as suas superfícies planas, tendo que adicionar diversas linhas para se aproximar de uma curva, deixando o modelo mais pesado.

# Escultura digital

A escultura digital é um modelo relativamente novo em relação aos outros dois. O método é parecido com uma modelagem artesanal de barro.

Através de uma forma como uma esfera por exemplo o artista pode ir modelando, exatamente como uma massa. Essa modelagem explora o lado mais criativo do artista, beneficiando também os detalhamentos do modelo. É utilizado principalmente na modelagem de personagens.

# Aplicação da modelagem 3D

A modelagem está presente em diversos campos do nosso dia a dia, com a digitalização do comércio e negócios o mercado 3D está se tornando cada vez mais popular.

## 1 – Impressão 3D

A impressão 3D oferece a possibilidade de trazer para o mundo real as modelagens digitais. Atualmente a impressão 3D teve um crescimento grande nos últimos anos, principalmente com a baixa dos custos de materiais e impressoras para impressão.

Alguns produtos que podem ser impressos são personagens, presentes, joias e moldes por exemplo.

A impressão 3D também funciona como testes de produtos em processo de desenvolvimento, chamado de prototipagem.

## 2 - Mercado de modelos

Os profissionais e empresas precisam de uma variedade de modelos para seus projetos, alguns deles demandam muito tempo no desenvolvimento, para sanar essa dificuldade há lojas online onde podemos adquirir modelos prontos de acordo com nossas necessidades, agilizando assim o processo.

## 3 – Outros usos

As modelagens podem ser utilizadas em diversos projetos como animações de filmes, desenvolvimento de jogos, arquitetura, indústria médica, cenários, artistas e muitas outras possibilidades.
