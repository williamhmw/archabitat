# Quixel Mixer - O que é e como utilizá-lo

![Quixel Mixer](/static/images/quixel-mixer-o-que-e-e-como-utiliza-lo/mixer-capa.png)

O software Quixel Mixer é uma ferramenta desenvolvida pela empresa Quixel, ela trabalha com a texturização de modelos 3D, combinando dados das imagens, pintura dos modelos em tempo real baseado em física (conhecido como PBR) e pintura procedural. Através do programa temos uma experiência simplificada com pouco passos para projetar os materiais nos modelos 3D.

Quixel Mixer está disponível gratuitamente para qualquer pessoa, para sempre, basta acessar a página da empresa, fazer o cadastro e efetuar o download.

# Facilidade de uso com Textura 3D fácil

A experiencia do usuário é simplificada, com o objetivo de tornar o software de texturização o mais fácil e acessível do mercado. Basta carregar um modelo 3D, aplicar ou dois materiais, ajustar as configurações e pronto, sua texturização está feita.

A curva de aprendizado foi montada para ser intuitiva e rápida para qualquer usuário que queira fazer a sua arte no modelo 3D.

![Quixel Mixer](/static/images/quixel-mixer-o-que-e-e-como-utiliza-lo/mixer-2.png)

# Diferenciais do programa

O Mixer possui uma abordagem específica para texturização dos modelos 3D, de modo personalizado algumas das ferramentas exclusivas são:

- Pintura 3D multicanal
- Dados de digitalização
- Curvatura 3D em tempo real
- Escultura de deslocamento
- Mascaramento de ID
- Ruídos procedimentais
- Projeção de textura contínua
- Materiais inteligentes

# Material inteligente

Essa ferramenta são várias camadas que você pode importar facilmente para o Mixer com um clique. Utilizando os recursos do software, o usuário pode ajustar cada camada em tempo real em qualquer modelo inserido.

![Quixel Mixer](/static/images/quixel-mixer-o-que-e-e-como-utiliza-lo/mixer-1.png)

Os dados extraídos de materiais importados da biblioteca do Megascans são ainda mais precisos, com parâmetros extraídos do mundo real e procedimentos dinâmicos.

Todos os materiais são finamente elaborados para serem totalmente personalizáveis como por exemplo a cor da superfície, quantidade de sujeira e tipo de material na superfície.
