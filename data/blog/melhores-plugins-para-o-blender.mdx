# Melhores plugins para o Blender

Quando estamos trabalhando em algum projeto, e precisamos de determinadas ferramentas que agilizem o trabalho, podemos recorrer aos plugins. No Blender podemos fazer a instalação dos plugins e utilizá-los nos projetos.

Para ajudar na seleção separamos uma lista com os melhores plugins para Blender.

---

<TOCInline toc={props.toc} asDisclosure />

---

# 1- fSpy

O fSpy é um software de códido aberto (open source) capaz de importar para o Blender as configurações e parâmetros extraídos de fotos, como pontos de fuga e direções perpendiculares de uma imagem.

Podemos importar para o aplicativo fotos sem dirtorções e aplicar os pontos de fugas diretamente no Blender, através do plugin de imposrtação do fSpy.

# 2 – Retopoflow 3

Este plugin é utilizado para retopologia, ele é um plugin completo que ajuda agilizar todo o processo de retopologia de um modelo 3D. Todas as ferramentas apresentadas no plugin são operadas demaneira muito rápida e prática com uma curva de aprendizado muito rápida.

Atualmente o plugin é vendido pela BlenderMarket com mais de 6.000 vendas.

# 3 – Hard-ops

O Hard-ops é um plugin para expandir o fluxo de trabalho booleano dentro do Blender de forma não-destrutiva ou destrutiva. Ele é extremamente indicado para as pessoas que desejam trabalhar com Hard Surface. Atualmente o plugin é vendido pela BlenderMarket.

# 4 – FLIP Fluids

O Plugin FLIP Fluids é um conjunto de ferramentas que auxiliam na configuração, execução e renderização de efeitos de fluídos como água por exemplo.Ele utiliza ténicas de simulação parecidas com as usadas nos softwares Houdini e Mantaflow por exemplo.

Com o longo período de desenvolvimento, o plugin possui uma excelente integração com o Blender, passando por processos de desenvolvimento beta com testadores e uma reputaçãode desenvolvimento de alta qualidade.

# 5 – BlenderKit

![Blenderkit](/static/images/melhores-plugins-para-o-blender/blenderkit_capa.jpg)

O BlenderKit permite baixar modelos 3D, pinceis e materiais diretamente dentro do Blender, sem limites de quantidades de ativos e de forma totalmente gratuita. São mais de 4.800 modelos disponiveis gratuitamente com uma alta qualidade.

Basta escolher um ativo porcategoria, selecionar, baixar e arrstar para a área de trabalho.

# 6 – Archpack

O Archpack é um conjunto de ferramentas demodelagem destinados a arquitetura, como foco na modelagem de construções como paredes, escadas, portas e janelas por exemplo.

O plugin possui uma versão gratuita open source e uma versão paga com mais recursos.

# 7 – Scatter

![Scatter](/static/images/melhores-plugins-para-o-blender/scatter_1_capa.png)

O Scatter é uma ferramenta de espalhamento, sendo possível multiplicar ativos como gramas, árvores e outros ativos que precisam de repetições de forma ordenada ou desordenada. Há recursos já prontos chamados de Present, como os biomas.
