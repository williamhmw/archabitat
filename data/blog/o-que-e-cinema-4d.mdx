# O que é Cinema 4D

![Cinema 4D](/static/images/o-que-e-cinema-4d/cinema-4d-capa.jpg)

O software de modelagem 3D Cinema 4D é desenvolvido atualmente pela empresa MAXON, o programa trabalha com texturização, iluminação, animação e renderização 3D. Pode ser usado para criar filmes, curtas metragens e modelagem para jogos. Abaixo confira uma listagem das vantagens do programa:

# Interface

A interface do programa é muito intuitiva e acessível para os iniciantes, todo o fluxo de trabalho é contínuo. Essa praticidade ajuda os profissionais da área a focarem no que realmente é importante, diminuindo o tempo despedido em outras tarefas. Há ainda a possibilidade de personalizar o layout do programa para cada usuário, se necessário poderá salvar o estilo da interface e carregá-la em outro lugar.

# Estabilidade

Um dos melhores pontos do Cinema 4D é a estabilidade do sistema. As novas funcionalidades que chegam a cada nova versão são testadas regularmente e passam por processos de qualidade que garantem menos bugs na versão de lançamento, além das atualizações para correção dos raros erros que podem vir a ter.

# Flexibilidade

Algumas soluções de trabalho, principalmente para grandes estúdio e empresas da área são o seu sistema. Há ferramentas como a modelagem paramétrica, shader procedural e texturização.

O cinema 4D possui ainda um render nativo para finalizar as produções, assim como a possibilidade de instalar plugins que adicionam novos recursos. Há também módulos específicos de motion graphics, que são animações complexas utilizadas pela indústria e integradas a softwares de edição de vídeo como o After Effects.

# Modelagem poligonal

A modelagem poligonal é a mais comum, encontrada em diversos softwares de modelagem tridimensional, ela tem como base polígonos, que são formas geométricas primitivas que a partir deles é possível construir objetos complexos.

# Modelagem procedural

A modelagem procedural é gerada por um algoritmo específico do programa na qual o artista define algumas regras e o software gera a forma desejada. Essa modelagem é muito útil em formas muitos complexas que demandariam tempo para serem modeladas, como árvores por exemplo.
